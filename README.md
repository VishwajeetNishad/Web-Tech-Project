# 🏠 NIET Hostel Management Dashboard

## 📌 Overview
The **NIET Hostel Management Dashboard** is a web-based application designed to simplify hostel and PG management. It helps administrators manage **students, rooms, payments, electricity usage, and notices** from a single dashboard.

This project is built using a **modular frontend architecture** with JavaScript, making it lightweight, fast, and easy to scale.

---

## 🚀 Features

- 📊 **Dashboard**
  - Overview of occupancy, payments, and activities  

- 👨‍🎓 **Student Management**
  - Add, update, and manage student records  
  - Track student history  

- 🏠 **Room Management**
  - Allocate rooms and monitor availability  

- 💳 **Payment Tracking**
  - Record rent payments  
  - Track pending dues  

- ⚡ **Electricity Management**
  - Track electricity usage and billing  

- 🔔 **Notices**
  - Send updates and announcements  

- ⚙️ **Settings**
  - Manage system configurations  

---

## 🛠️ Tech Stack

- **Frontend:** HTML, CSS, JavaScript  
- **Icons:** Font Awesome  
- **Architecture:** Modular JavaScript
